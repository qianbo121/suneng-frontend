### 页面结构判断

- 一级栏目基本全部采用：`Banner -> 面包屑 -> 横向子导航 -> 主内容区 -> 页脚`
- 产品中心是唯一明确出现左侧竖向侧栏导航的栏目，且详情页延续此结构
- 关于我们、企业实力、服务支持、合作伙伴、新闻中心、联系我们都更接近“顶部 Banner + 内容页签”的结构，而不是左侧树导航

## B. 全站公共结构

### 1. 顶部区域

- 截图未见独立“顶部通栏 + 主导航”双层结构。
- 页面最上方就是白底主导航，承担品牌、栏目、语言切换入口。
- 高度目测约 `88px - 96px`。
- 背景色为纯白，底部无明显描边，依靠与 Banner 的色差分层。

### 2. 主导航

- Logo 位于左侧，横向排布，留白较大。
- 主菜单位于中部偏右，单行排列，文本为深色粗体。
- 右侧为语言切换区，固定为整块红底按钮，宽度约 `110px - 130px`。
- 当前一级栏目采用整块红底高亮，文字转白。
- 非当前栏目 hover 时，以文字色变化或下拉展开为主，没有观察到复杂动画。

### 3. 二级导航 / 下拉菜单

- 触发方式已确认是 `hover`。
- 下拉面板为白底竖排菜单，宽度约 `200px - 220px`。
- 每个二级项之间有浅灰分隔线。
- 面板阴影很轻，重心在纯净留白，不走厚重投影。
- 动画建议采用 `fade + translateY(6px)`，因为截图显示其风格偏克制。

### 4. 移动端导航

- 未提供移动端截图，不能把任何具体样式当作对标事实。
- 开发建议保留“汉堡菜单 -> 右侧抽屉/全屏抽屉 -> 一级折叠 + 二级展开”的模式。
- 抽屉内应保留语言切换、联系电话和核心 CTA。
- 该部分需要你补充移动端截图后再做高保真定稿。

### 5. 页脚

- 深灰背景，大区块式信息页脚。
- 左侧为品牌信息、服务热线、地址、社交图标。
- 中部为多列站点链接，至少包含：企业概况、产品中心、企业实力、服务中心、合作伙伴、新闻中心。
- 右侧为两个二维码位。
- 最底部有版权/备案/建站信息与 IPv4/IPv6 支持文案。
- 页脚整体分为两层：主信息层 + 版权层，中间用细线分隔。

### 6. 右侧浮动工具栏

- 已确认存在固定在右侧中部的白底悬浮按钮组。
- 单个按钮为圆角矩形白卡，阴影极轻。
- 已能确认的功能包括：邮箱、电话/咨询、返回顶部、客服入口。
- hover 会向左展开红底提示条，例如“企业邮箱：xxx@xxx.com”。
- 长截图出现了多个重复按钮，这是截图工具对 fixed 元素的重复采样，不应按多个工具组实现。
- 移动端建议仅保留电话、在线咨询、返回顶部 3 个高频入口；具体形态需待移动端截图确认。

### 7. 内页 PageBanner

- 全宽大图 Banner，图像风格为企业场景、设备、厂房、服务或合作场景。
- 左下或左中区域叠加栏目标题与副标题。
- 标题为白色大字，副标题为较小白字。
- Banner 高度目测约 `260px - 340px`。
- Banner 下方紧接独立的面包屑条，再下方是子导航标签。

### 8. 面包屑逻辑

- 基础模式：`首页 > 一级栏目 > 二级栏目/当前页`
- 详情页模式：`首页 > 一级栏目 > 分类/列表 > 详情标题`
- 面包屑位于 Banner 下方白底区域，左对齐，字号较小，颜色偏浅灰。

### 9. 侧边栏导航

- 已确认仅产品中心采用左侧竖向导航。
- 结构分为：栏目标题块、分类列表、联系方式模块。
- 当前分类为红底白字。
- 详情页中左侧会堆叠多个分类分组，说明产品中心信息架构较深，建议 CMS 支持多级产品分类。

### 10. 通用组件规律

- 按钮：以红色描边按钮和红色实心胶囊按钮两种为主。
- 标题：英文小标题 + 中文大标题，或浅色大号英文描边字 + 红色中文标题。
- 卡片：整体方正，圆角较小或无圆角，偏企业型。
- 列表：新闻列表采用左图右文；合作企业采用 Logo 宫格；荣誉/证书/设备采用图片宫格。
- 分页：小尺寸方形分页，当前页红底白字，其他页浅灰边框。

## 2. 逆向结论

- 该站是典型制造业官网，但信息组织并不复杂，核心是“品牌背书 + 产品展示 + 资质荣誉 + 新闻 + 联系转化”。
- 前端首轮完全可以用 `mock data` 做到高保真；真正影响后续成本的是 CMS 模型是否一次性拆对。
- 关于我们、企业实力、服务支持、合作伙伴、新闻中心都使用同一套内页壳层，后续组件复用率会很高。
- 产品中心是全站信息密度最高的模块，必须从第一阶段就按“分类树 + 产品详情 + 参数表 + 图纸 + 咨询表单”来设计。
- 当前最需要你补充的不是代码，而是缺图页和移动端截图；缺这些会直接影响高保真还原准确度。
